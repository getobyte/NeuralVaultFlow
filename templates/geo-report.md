# GEO Report Template

Template for NVC key `{project}:geo-audit-{date}` — Generative Engine Optimization audit reports with scoring and prioritized recommendations.

**Purpose:** Consistent GEO audit output that quantifies AI search visibility and provides actionable optimization steps.

## Field Documentation

| Field | Type | Required | Purpose | Example |
|-------|------|----------|---------|---------|
| `{project-name}` | variable | Yes | Project/site name | `acme-corp` |
| `{date}` | variable | Yes | Audit date | `2026-03-26` |
| `{site-url}` | variable | Yes | Primary URL | `https://acme.com` |
| `{business-type}` | variable | Yes | Type of business | `SaaS`, `E-commerce`, `Agency` |
| `{total-score}` | variable | Yes | Weighted GEO score | `67` |
| `{rating}` | variable | Yes | Score interpretation | `Well Optimized` |
| `{score}` | variable | Yes | Per-category raw score | `75` |
| `{weighted}` | variable | Yes | Score × weight | `18.75` |
| `{impact-level}` | variable | Yes | Quick win impact | `High`, `Medium` |
| `{effort-level}` | variable | Yes | Implementation effort | `Low`, `Medium`, `High` |

## Section Specifications

### GEO Score Table
**Purpose:** At-a-glance scoring per the formula in `frameworks/geo-scoring.md`.
**Quality check:** Weighted scores sum to total. Rating matches score range.

### Quick Wins
**Purpose:** Top 3 highest-ROI improvements.
**Quality check:** Each has current state, target state, and actionable steps. Ordered by impact/effort ratio.

### Category Details
**Purpose:** Detailed breakdown justifying each category score.
**Quality check:** Every score below 60 has specific findings and actions.

### Action Plan
**Purpose:** Prioritized roadmap for all improvements.
**Quality check:** Actions are specific. Estimated point impact is realistic.
